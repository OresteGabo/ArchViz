## üåü The Vision
The ArchViz project was born from a real-world frustration: technical plans and 3D renders on a screen fail to convey the **true sense of space**. As a non-engineer, I realized that looking at a layout isn't the same as living in it. 

This platform allows users to bring their **exact physical dimensions** into a digital twin of their future home to answer critical ergonomic questions:
- Can I reach the top shelf of this cabinet?
- Is this doorway wide enough for my comfort?
- Does the room's ergonomics match my morphology?

## üõ†Ô∏è Key Technical Features
- **Biometric Viewport Scaling**: Adjusts the XR camera height based on real-world user dimensions (Adult, Child, etc.) for accurate ergonomic testing.
- **1:1 Scale Locomotion**: Synchronized movement system to ensure that physical steps match virtual displacement.
- **Multi-Source BIM Integration**: Architected to support runtime imports from **Revit, ArchiCAD, Blender, FBX, and glTF**.
- **Modular C# Architecture**: Built with a decoupled, interface-based design (SOLID principles) for high scalability and clean code.
